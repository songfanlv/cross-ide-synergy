# Antigravity Share - 协同编辑扩展 (跨局域网开源版)

> 实时协同编辑扩展，支持 VS Code 和 Antigravity 编辑器。全新版本引入了 **跨局域网 (WAN) 协同** 功能，让异地开发者也能通过分享链接完美、低延迟地编辑同一个项目。

## ✨ 核心特性

- **跨局域网协同 (WAN) [新增]** — 支持通过 EMQX 中继服务器实现跨越局域网的公网协同编辑，稳定可靠。
- **局域网极速直连 (LAN)** — 在同一内网下，自动使用本地直连，体验极致打字同步。
- **一键分享协作** — 轻松生成专属协作链接 (`ag-share://`)，告别繁琐的账号配置。
- **全自动工作区同步** — 自动将主机的当前全部工作区文件完整同步给新加入的访客。
- **防循环广播机制** — 内置 Mutex 互斥锁完美解决极端并发情况下的死循环更新和崩溃问题。
- **断线无痕清理** — 作为访客模式产生的临时文件，在断开连接后即刻销毁，始终保持本机环境纯净。

## 📦 安装说明

我们在本仓库中提供了预打包的 `.vsix` 插件文件，您可以直接安装使用：
1. 下载本项目文件夹内的 `antigravity-share-v3.vsix`（位于当前目录）
2. 打开 VS Code 或者 Antigravity 编辑器，按下快捷键 `Ctrl+Shift+P`
3. 搜索并执行 `Extensions: Install from VSIX...`
4. 选择刚才下载的 VSIX 文件即可完成安装

## 🚀 快速上手

### 跨局域网协同配置 (可选但推荐)
1. 在编辑器的设置中搜索 `Antigravity Share` 或者 `Relay Url`
2. 找到 `Relay Url` 配置项，如果有自定义的 WebSocket 代理或中继可以填入。
3. 如果该项留空，插件将默认底层使用稳定的公用 EMQX 中继以实现穿透。

### 分享者 (Host)
1. 使用编辑器打开任意您需要协作分享的工作区文件夹
2. 按 `Ctrl+Shift+P` 运行 **`Antigravity: 分享项目 (Share Project)`**
3. 自动生成并复制弹出的协作链接
4. 将该链接通过任意方式发送给协作者

### 协作者 (Guest)
1. 打开一个空的工作区或让扩展自动生成临时文件夹
2. 按 `Ctrl+Shift+P` 运行 **`Antigravity: 加入协作 (Join Collaboration)`**
3. 粘贴分享者发送过来的 `ag-share://` 链接
4. 连接成功后，主机的全部工作区文件将自动全量推送同步，开始实时的远程协同编程！

## 🛠️ 本地开发与专属中继部署

如果您希望基于本项目进行修改、增添新功能或参与维护：

```bash
# 1. 确保已安装 Node.js，然后安装项目所有的依赖
npm install

# 2. 启动编译监听模式
npm run watch

# 3. 在编辑器 (VS Code) 中按下 F5，启动「扩展调试宿主」体验调试开发
```
