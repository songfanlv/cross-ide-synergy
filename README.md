# Antigravity Share (局域网纯净版 V1)

Antigravity Share 是一个轻量级、无依赖的 VS Code 实时协同编辑插件，专为**同网段（局域网）**或同一设备内的小型团队协作开发设计（后续会继续攻克跨局域网连接）。它没有复杂的云端服务器或内网穿透逻辑，数据仅在本地网络流转，保证绝对的安全与高速度。

## 🌟 功能特色
* **零配置局域网同步**：仅需点击一次即可启动 Host，同网段其他协作者通过 IP 地址立即接入。
* **文件和光标实时同步**：支持文件的增删改、实时的光标和选中区域高亮显示。
* **极速响应**：完全基于 Node.js 原生的 WebSocket 实现，摒弃第三方穿透，同局域网内延迟几近为零。
* **纯净安全**：没有外部请求、没有数据上报，代码资产仅在协作者的电脑之间流转。

## 🚀 适用场景
* 团队会议室：大家连着同一个会议室 WiFi，主讲人电脑开启分享，其他同学实时查看代码修改。
* 企业内网：不便连接外网、高密级项目的内部同网段协同开发。

## 📥 安装指南
由于此版本专注于局域网，未上架官方市场，您可以通过 VSIX 文件进行离线安装：
1. 在 Github Releases 页面下载 `antigravity-share-v1.0.0.vsix`。
2. 打开 VS Code，进入“扩展”面板 (Ctrl+Shift+X)。
3. 点击面板右上角的 `...` 菜单，选择 **"从 VSIX 安装..."** (Install from VSIX...)。
4. 选中刚刚下载的文件即可完成安装。

## 📖 使用说明

### 👨‍💻 角色一：分享者 (Host)
1. 在 VS Code 中打开您需要分享的**项目文件夹**。
2. 点击底部状态栏左侧的 `$(broadcast) Share` 按钮，或按 `Ctrl+Shift+P` 运行命令 `Antigravity: 分享项目`。
3. 允许弹出提示，插件将自动获取您的**局域网IP**和**连接端口**，并生成协作链接（如 `ag-share://192.168.1.100:xxxxx`）。
4. 将该链接通过微信/企业微信等任何方式发送给您的协作者。

### 👨‍💻 角色二：接入者 (Guest)
1. 在 VS Code 中打开一个**完全空**的新文件夹（用于接收 Host 同步的项目文件）。
2. 按 `Ctrl+Shift+P` 运行命令 `Antigravity: 加入协作`。
3. 粘贴分享者发来的链接（注意必须处于同一个局域网能够互通）。
4. 等待几秒钟，文件树将自动同步展现，随时可以与分享者开始协同编写！

## ⚠️ 局域网版使用限制
1. **网络要求**：如果 Guest 电脑与 Host 电脑不在同一个内网、无法 `ping` 通对方，将无法建立连接。
2. **端口拦截**：部分企业电脑的防火墙可能会拦截随机端口。如果连接失败，请确保操作系统的本地防火墙允许 VS Code 或以 Node.js 建立传入和传出连接。
3. **保持运行**：协同编辑期间，分享者 (Host) 必须保持 VS Code 及项目目录处于开启状态。关闭 VS Code 将立即断开并停止服务。
